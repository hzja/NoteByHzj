# 2、下载安装busybox
```bash
$ wget https://www.busybox.net/downloads/binaries/1.30.0-i686/busybox
$ chmod +x busybox
```
<a name="9vg3u"></a>
# 3、运行busybox
下面的命令提供进程的实时视图，每两秒显示进程的状态信息，使用上下键可以查看更多
```bash
$ ./busybox top -d 2
```
