### 2、Starship
Starship 可以适用于任何 Shell。只需要安装它，然后在相应的配置文件`.bashrc/.zshrc/config.fish`添加一行配置，剩下的工作交给它来完成就好了。<br />![](https://cdn.nlark.com/yuque/0/2021/webp/396745/1636421043090-1d956f12-4ba4-4996-b791-e33c362376bc.webp#clientId=u0ba4951b-f03b-4&from=paste&id=uf5dab8b9&originHeight=688&originWidth=1080&originalType=url&ratio=1&status=done&style=none&taskId=ue8c3e1b5-c046-4e0c-b997-5d0095aafb7)<br />它可以做到：

- 根据是否在代码仓库中添加了新文件、是否修改了文件、是否暂存了文件等情况，用相应的符号表示 git 仓库的状态。
- 根据所在的 Python 项目目录，展示 Python 的版本号，这也适用于 Go/Node/Rust/Elm 等其他编程语言环境。
- 展示上一个命令执行所用的时间，指令运行时间必须在毫秒级别。
- 如果上一个命令执行失败，会展示相应的错误提示符。

### 3、z
“z” 可以快速地在文件目录之间跳转。它会记住访问的历史文件夹，经过短暂的学习后，就可以使用`z path_of_the_folder_name`命令在目录之间跳转了。<br />![](https://cdn.nlark.com/yuque/0/2021/gif/396745/1636421043085-e34a3dd5-2661-4cfc-ae1f-098067a1a72c.gif#clientId=u0ba4951b-f03b-4&from=paste&id=u8d4c4823&originHeight=436&originWidth=631&originalType=url&ratio=1&status=done&style=none&taskId=uba14b3a6-5534-4b16-9e5f-504173cdfc6)<br />比如，如果经常访问 `~/work/src/projects`，只需要运行 `z pro` ，就可以立马跳转到那里。z 的原理参考了 frecency 算法——一个基于统计 frequency 和 recency 进行分析的算法。如果它存储了不想使用的路径文件夹，随时可以手动将其删除。它提高了在常用的不同文件路径之间频繁切换的效率，节省了键盘击键次数以及大量的路径记忆。<br />下载地址：[https://github.com/rupa/z](https://github.com/rupa/z)
