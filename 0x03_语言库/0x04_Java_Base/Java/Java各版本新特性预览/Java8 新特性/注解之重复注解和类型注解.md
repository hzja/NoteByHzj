### 一、可重复注解
<a name="OC14t"></a>
#### A.定义注解
```java
import java.lang.annotation.Repeatable;
import java.lang.annotation.Retention;
import java.lang.annotation.RetentionPolicy;
import java.lang.annotation.Target;

/**
 * annotation
 * <p>
 * encoding:UTF-8
 *
 * @author Fcant 下午 22:10:52 2020/2/23/0023
 */
@Repeatable(Annotations.class)
@Target({TYPE, FIELD, METHOD, PARAMETER, CONSTRUCTOR, LOCAL_VARIABLE})
@Retention(RetentionPolicy.RUNTIME)
public @interface Anno {
    String value() default "Fcant";
}
```
<a name="Rk3tZ"></a>
#### B.定义注解容器
```java
import java.lang.annotation.Retention;
import java.lang.annotation.RetentionPolicy;
import java.lang.annotation.Target;

import static java.lang.annotation.ElementType.*;

/**
 * Annotations
 * <p>
 * encoding:UTF-8
 *
 * @author Fcant 下午 22:16:43 2020/2/23/0023
 */
@Target({TYPE, FIELD, METHOD, PARAMETER, CONSTRUCTOR, LOCAL_VARIABLE})
@Retention(RetentionPolicy.RUNTIME)
public @interface Annotations {
    Anno[] value();
}
```
<a name="a7f8fd52"></a>
#### C.测试使用
```java
import org.junit.Test;

import java.lang.reflect.Method;

/**
 * TestAnnotation
 * <p>
 * encoding:UTF-8
 *
 * @author Fcant 下午 22:15:22 2020/2/23/0023
 */
public class TestAnnotation {

    @Test
    public void annotationTest() throws NoSuchMethodException {
        Class<TestAnnotation> annotationClass = TestAnnotation.class;
        Method show = annotationClass.getMethod("show");
        Anno[] annotationsByType = show.getAnnotationsByType(Anno.class);
        for (Anno anno : annotationsByType) {
            System.out.println(anno.value());
        }
    }

    @Anno("Hello")
    @Anno("World")
    public void show() {

    }
}
```

### 二、类型注解
<a name="ZLMEI"></a>
#### A.修改定义的类型注解
```java

 * annotation
```
