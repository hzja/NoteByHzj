Java 爬虫
<a name="peldR"></a>
## 介绍
平台以流程图的方式定义爬虫,是一个高度灵活可配置的爬虫平台
<a name="Rhte1"></a>
## 特性

- 支持Xpath/JsonPath/css选择器/正则提取/混搭提取
- 支持JSON/XML/二进制格式
- 支持多数据源、SQL select/selectInt/selectOne/insert/update/delete
- 支持爬取JS动态渲染(或ajax)的页面
- 支持代理
- 支持自动保存至数据库/文件
- 常用字符串、日期、文件、加解密等函数
- 支持插件扩展(自定义执行器，自定义方法）
- 任务监控,任务日志
- 支持HTTP接口
- 支持Cookie自动管理
- 支持自定义函数
<a name="fypnK"></a>
## 插件

- Selenium插件
- Redis插件
- OSS插件
- Mongodb插件
- IP代理池插件
- OCR识别插件
- 电子邮箱插件
