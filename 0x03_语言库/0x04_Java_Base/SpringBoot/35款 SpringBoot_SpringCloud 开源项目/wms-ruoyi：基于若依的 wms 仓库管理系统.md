## 项目介绍
- 从项目名称就可以看出来，这是一套基于若依的 wms 仓库管理系统，支持仓库基础数据维护、入库、出库、库存看板等功能。
- 个人及企业免费使用，完全开源。
<a name="HFcz9"></a>
## 技术栈

- 前端：Vue+Element UI（ant design 正在开发中）
- 后端：Spring Boot + Spring Security + Jwt + MySQL + Redis
