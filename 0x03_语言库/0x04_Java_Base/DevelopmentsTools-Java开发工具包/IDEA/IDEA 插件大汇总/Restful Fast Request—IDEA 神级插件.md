## **功能一览**

- API 导航树
- SearchEveryWhere 搜索 API
- API 导出到 Postman
- 一键生成 url 和参数
- 发送请求调试 API(支持修改生成的参数)
- 域名定制及切换
- 自定义参数解析
- 字符串生成策略化
- 保存 API
- 管理 API(搜索过滤、修改 API 名称)
- 一键定位历史保存的 API 到方法
- API 转 curl
- 自动将 API 关联到模块分组
- 内置 Json 参数编辑器
