<a name="wGAV1"></a>

### 两种模式区别
不同模式下cell前颜色会变化，见上面两图红圈部分；<br />edit mode（编辑模式）下，可直接在cell中写代码；<br />command mode（命令模式）下可以使用一些快捷键;<br />下文介绍时，快捷键前添加了Esc为command mode。
