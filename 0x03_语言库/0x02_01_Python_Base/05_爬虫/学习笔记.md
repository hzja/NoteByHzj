## 数据识别

+ 网页源代码可以看到的就是静态加载

+ 网页源代码看不到的而点击网络->xhr后可看到数据的话就是动态加载

+ 方式不同：
  + 静态，可直接访问查询的URL地址 
  + 动态，直接访问找到的接口地址
